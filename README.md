# Yandex Mail Bot

Бот для пересылки писем из Яндекс.Почты в чат ВКонтакте.

## Установка

```bash
pip install vk-api python-dotenv
```

## Настройка

1. Скопируй `.env.example` в `.env`
2. Заполни переменные:

### Яндекс.Почта

1. Включи IMAP в настройках почты: Настройки → Почтовые программы
2. Создай пароль приложения: https://id.yandex.ru/security/app-passwords
3. Укажи email и пароль приложения в `.env`

### ВКонтакте

1. Получи токен: https://vkhost.github.io/
2. Укажи `peer_id`:
   - Для личных сообщений: ID пользователя
   - Для беседы: `2000000000 + номер беседы`

## Запуск

```bash
python main.py
```

## Режимы работы

### IDLE (по умолчанию)
```
MODE=idle
```
Бот использует IMAP IDLE — сервер сам уведомляет о новых письмах. Мгновенные уведомления без постоянных запросов.

### Polling
```
MODE=polling
POLL_INTERVAL=30
```
Проверка почты каждые N секунд. Используй, если IDLE не работает.

## Что отправляется в ВК

- Тема письма
- Отправитель
- Дата
- Полное содержимое

Длинные письма автоматически разбиваются на части.
